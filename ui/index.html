Entities
Global entity-class filter applied before anything else in the helper processes the live entity list — affects the trees on the Game tab, the radar, exports / dumps, and the AutoPilot's target picker. Unchecking a category hides that whole class of entities everywhere; if something is missing from the radar that you'd expect to see, check here first.
Skills & Buffs Blacklist
Skill / buff IDs added here are suppressed from the Skills & Buffs tree and from the overlay's status text — useful for hiding constant-up clutter like aura icons, infinite-duration debuffs, or self-applied cosmetic effects. Add an entry via the ➕ button next to a buff on the Skills tab; remove with the ✕ button next to its name below.
